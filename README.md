# RTS Lab2.1-2.2

## Usage

1. Clone the repo and move into the cloned directory
2. Install dependencies: `$ yarn install` (`$ npm install`, if you use npm)
3. Execute with `$ yarn start` (`$ npm start`, if you use npm)
